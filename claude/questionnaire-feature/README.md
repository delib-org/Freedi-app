# Questionnaire Feature

## Overview
This feature enables the creation and management of questionnaires with multiple sequential questions.

## Feature Description
- Ability to create questionnaires with multiple questions in sequence
- Users can respond to questions one after another
- Questionnaire results can be collected and analyzed
